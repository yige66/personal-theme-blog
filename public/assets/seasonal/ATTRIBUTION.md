## Seasonal effect assets

Downloaded CC0 source assets used by the seasonal background effects:

- Kenney Particle Pack, Creative Commons CC0 1.0: https://www.kenney.nl/assets/particle-pack
- Kenney Foliage Sprites, Creative Commons CC0 1.0: https://www.kenney.nl/assets/foliage-sprites
- Paramecijs vegetation base texture pack, Public Domain/CC0: https://opengameart.org/content/paramecijs-vegetation-base-texture-pack
- Para CC0 Particle FX 2, CC0: https://opengameart.org/content/para-cc0-particle-fx-2
- Flares, CC0/Public Domain: https://opengameart.org/content/flares
- Y2K Ice Texture, CC0/Public Domain: https://opengameart.org/content/y2k-ice-texture

The project only includes selected PNG files from these packs for seasonal petals, particles, light beams, heat haze, firefly glow, snow, and autumn leaves.
